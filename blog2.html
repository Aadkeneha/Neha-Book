The Learnings

The capacity to learn is a gift; the ability to learn is a skill; the willingness to learn is a choice.
- by Brian Herbert.

Learning is a constant journey which is probably the endless one. My learning journey of web development started with my first college project. I started with the most basic but important language in web development which is HTML. HTML stands for HyperText Markup Language. This language serves as the structure of a website, it helps in making the basic backbone.

Then we have another important language, CSS which stands for Cascading Style Sheets. This is a stylesheet language that decides how the website will look on the web browser to the users. It helps in styling the HTML elements and making the website more beautiful in an overall view.

The last for today, we have JavaScript. This serves as a script that helps us in performing various actions on a website and thus making the website interactive.
These 3 languages are the required ones which are enough for building wonderful websites. These were my learning's introduction in brief form.
